Convolutional networks ("CNNs") are a special kind of neural nets that use several clever tricks to learn faster and better. ConvNets essentially revolutionized computer vision and are heavily used in speech recognition and text classification as well.

Convolutional networks are implemented in every major framework. It is usually easier to understand the code that is written using higher level libraries.
* [Theano: Convolutional Neural Networks (LeNet)](http://deeplearning.net/tutorial/lenet.html)
* [Using Lasagne for training Deep Neural Networks](http://luizgh.github.io/libraries/2015/12/08/getting-started-with-lasagne/)
* [Detecting diabetic retinopathy in eye images](http://jeffreydf.github.io/diabetic-retinopathy-detection/) - a blog post by one of the best performers of Diabetic retinopathy detection contest in Kaggle. Includes a good example of data augmentation.
* [Face recognition for right whales using deep learning](https://blog.deepsense.ai/deep-learning-right-whale-recognition-kaggle/) - the authors used different ConvNets for localization and classification.
* [Tensorflow: Convolutional neural networks](https://www.tensorflow.org/tutorials/deep_cnn) for image classification on CIFAR-10 dataset.
* [Implementing a CNN for text classification in Tensorflow](http://www.wildml.com/2015/12/implementing-a-cnn-for-text-classification-in-tensorflow/)
* [DeepDream implementation in TensorFlow](https://nbviewer.jupyter.org/github/tensorflow/tensorflow/blob/master/tensorflow/examples/tutorials/deepdream/deepdream.ipynb)
* [92.45% on CIFAR-10 in Torch](http://torch.ch/blog/2015/07/30/cifar.html) - implements famous VGGNet network with batch normalization layers in Torch
* [Training and investigating Residual Nets](http://torch.ch/blog/2016/02/04/resnets.html) - Residual networks perform very well on image classification tasks. Two researchers from Facebook and CornellTech implemented these networks in Torch
* [ConvNets in practice](https://www.youtube.com/watch?v=ue4RJdI8yRA&list=PLlJy-eBtNFt6EuMxFYRiNRS07MCWN5UIA&index=11) - lots of practical tips on using convolutional networks including data augmentation, transfer learning, fast implementations of convolution operation
